フランス語の単語や文を正しく発音するための鉄板ルール（その１）<!--more-->

### １．フランス語では末尾の子音字は原則発音しない。
例外的に発音する単語もあるため、原則となります。
ついつい英語読み末尾の子音字を発音したくなる単語を以下に示します。

{% capture notice-1 %}
**Example**:
- grand (グラン)「大きい」
- chat　(シャ)「猫」
- fort (フォール)「強い㊚」
- moment (モマン)「一瞬」
- appartement (アパルトマン)「アパートメント」
- port (ポール)「港」
- placard (プラカール)「戸棚」
- petit (プティ)「小さい」
{% endcapture %}

### ２．「c」、「f」、「l」、「r」の文字は基本的に発音する

{% capture notice-1 %}
**Example**:
- avec (アヴェク)「〜ともに」
- parc (パルク)「公園」
- sac (サック)「かばん」
- actif (アクティフ)「活発な
- chef (シェフ)「上司」
- péril (ぺリル)「危険」
- sel (セル)「塩」
- cher (シェール)「高価な」
- mer (メール)「海」
- or (オール)「金：ゴールド」
- bus (ブゥス)「バス」
{% endcapture %}
<div class="notice--info"><span style="font-size:1.2em;">{{ notice-1 | markdownify }}</span></div>

### ３．複数形の s は例外なく発音しない

単数形なのか複数形なのか聞いただけでは区別できません。冠詞・所有格で「後ろの」名詞の単複を判別できます。

{% capture notice-1 %}
**Example**:
- ami (アミ ㊚単)「男友達」　amis (アミ　複)「友達たち」
- amie(アミ ㊛単)「女友達」

- mon ami(モナミ)「私の恋人㊚」　mes amis (メザミ)「私のすべての友達たち」
- mon amie(モナミ)「私の恋人㊛」

- un ami(アンナミ)「一人の男友達」 des amis (デザミ)「友達たち」
- une amie(ユナミ)「一人の女友達」
{% endcapture %}

### ４．h は発音しない

h 自体は、どのような場合でも発音しない。

{% capture notice-1 %}
**Example**:
- hotel (オテル)「ホテル」
- homme (オム)「男のひと」
- rhume (リュム)「風邪」
- huile (ユイル)「油」
- heure (ウール)「時間」
- horaire (オレール)「時間割」
- hier (イエール)「昨日」
- habiter (アビテ)「住む」
- habitude (アビテュド)「習慣」
- huit (ユイット)「８」
- haut / haute (オー / オート)「高い」
- habiller (アビエ)「着せる」
- heureusement (ウルーズマン)「運よく」
- hiver (イヴェール)「冬」
- dehors (ドゥオール)「外に，外で」
- aujourd'hui (オジュルデュイ)「今日」
- histoire (イストワール)「歴史」
{% endcapture %}
