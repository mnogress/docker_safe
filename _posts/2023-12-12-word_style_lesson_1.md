"Word スタイル"は、テキストや段落の書式設定を一括で管理するための機能です。
"スタイル"は、テキストのフォント、サイズ、色、段落のインデント、行間などの書式設定を含んでいます。これにより、
文書全体で一貫性のあるデザインを簡単に維持したり、スムーズに変更したりすることができます。

Word スタイルを使用すると、特定のテキストに対して手動で書式を設定する必要がなくなります。
代わりに、見出し、本文、引用などの異なる要素に対してあらかじめ定義されたスタイルを適用することができます。
文書全体の書式を変更する場合、スタイルの定義を変更するだけで、その変更が文書全体に反映されます。

スタイルの使用は、効率的な文書作成や一貫性の維持に役立ちます。特に大規模な文書やチームでの共同作業の際に便利です。

</div>

    padding: 15px 10px 7px;
    border: solid 2px #DF8914;
    border-radius: 6px;
}
.box28 .box-title {
    position: absolute;
    display: inline-block;
    top:  -15px;
    left:  15px;
    padding: 0 9px;
    height: 35px;
    line-height: 35px;
    font-size: 20px;
    background: #DF8914;
    color: #ffffff;
    font-weight: bold;
    border-radius: 6px;
}
.box28 p {
    margin: 0; 
    padding: 1em;
}
</style>
