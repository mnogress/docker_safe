このページでは、フランス語の身体の部位を表す語彙をまとめています。日常会話、病院での説明、自己紹介など、さまざまな場面で役立つ基本語彙です。
日本人学習者が間違えやすいポイントもあわせて解説します。
<!--more-->


### 身体語彙の特徴
フランス語の身体語彙には、いくつか特徴があります。

<main class="page-ul-fix">
<ul>
<li>髪（les cheveux）は常に複数形</li>  
<li>目は単数と複数で形が変わる（un œil / les yeux）</li>  
<li>手（la main）は女性名詞</li>  
<li>身体の部位には所有形容詞ではなく定冠詞を使う（Je me lave les mains）</li>
</ul>
</main>

### 日本語話者が特につまずきやすいポイント

<div class="box33">
    <span class="box-title">よくある間違い</span>
    <ol>
    <li>身体の部位には所有形容詞を使わない</li>
    <ul>
    <li>私の手を洗う  〇 Je me lave <strong>les</strong> mains.   ✕ Je me lave <strong>mes</strong> mains.</li>
    <li>彼は頭が痛い  〇 Il a mal <strong>à la</strong> tête.   ✕ Il a mal <strong>à sa</strong> tête.</li>
    <li>彼女は足を折った  〇 Elle s’est cassé <strong>la</strong> jambe.  ✕ Elle s’est cassé <strong>sa</strong> jambe.</li>
    </ul>
    <li>髪（les cheveux）は常に複数形</li>
    <li>un œil / les yeux の形の違い</li>
    <li>指（le doigt）の綴りは特殊</li>
    <li>親指（le pouce）は「インチ」の意味もある</li>
    </ol>
</div>

定冠詞が　l' になりますので、㊚，㊛　の記号を追加しています。

<section aria-labelledby="glossary-title">
  <h3 id="glossary-title">一緒に覚えたい語句・用法</h3>
  <dl class="glossary">
    <dt>saigner</dt>
    <dd>セニェ 出血する</dd>
    <dt>le comportement</dt>
    <dd>ル・コンポルタモン，振る舞い；態度</dd>
    <dt>J'ai le nez qui coule.</dt>
    <dd>ジェ・ルネ・キクル　鼻水がでる (running nose) そのまま覚える</dd>
    <dt>s'est blessé</dt>
    <dd>「怪我をする、負傷する」の代名動詞 se blesser の複合過去形</dd>
    <dt>s'est cassé</dt>
    <dd>「壊す、折る」の代名動詞 se casser の複合過去形</dd>
    <dt>essuyer</dt>
    <dd>エスュイエ　拭く　->　s'essuyer 自分の体を拭く</dd>
  </dl>

document.addEventListener('DOMContentLoaded', () => {
  document.querySelectorAll('.player').forEach((wrap, idx) => {
    const audioEl = wrap.querySelector('audio');
    const btn = wrap.querySelector('button');

    console.log(`player #${idx+1}`, { audioEl, btn });

    if (!audioEl || !btn) {
      console.warn(`player #${idx+1} に audio または button が見つかりません`);
      return;
    }
    btn.addEventListener('click', () => {
      if (audioEl.paused) audioEl.play().catch(err => console.error('再生失敗:', err));
      else audioEl.pause();
    });

    audioEl.addEventListener('ended', () => updateLabel(btn, false));
    audioEl.addEventListener('play',  () => updateLabel(btn, true));
    audioEl.addEventListener('pause', () => updateLabel(btn, false));
  });

  function updateLabel(button, playing) {
    const label = button.querySelector('.label');
    if (label) label.textContent = playing ? '停止' : '🔄';
  }
});
